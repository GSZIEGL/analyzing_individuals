# Játékos-összehasonlító Streamlit app

## Indítás
```bash
streamlit run app.py
```

## Mire képes
- Excel alapú 2 játékos összehasonlítás
- Posztcsoport választás
- Posztspecifikus mutatók szűrése
- Pókháló fix 0–100 skálán
- Nyers összehasonlító oszlopdiagram
- Rövid szöveges erősség/gyengeség összefoglaló
- Letölthető összehasonlító táblázat

## Elvárt Excel-struktúra
Az app a most feltöltött összehasonlító Excel mintájára épül:
- a 3. és 4. oszlopban legyen a két játékos
- az első oszlopban a mutatónevek
- a Position sor tartalmazza a posztot
